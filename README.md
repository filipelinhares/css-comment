# CSS/SASS Comments pattern

## CSS

### Header of a section

ch `⇥` expand to:
```css
/* # =================================================================
   # My section title
   # ================================================================= */
```

### Subsection

cs `⇥` expand to:
```css
/* # My subsection title
   # ================================================================= */
```

### Divider

cd `⇥` expand to:
```css
/* # ========== My divider */
```

### Simple comment

c `⇥` expand to:
```css
/* # A simple declaration */
```

### Simple documentation with a detailed explanation

cdo `⇥` expand to:
```css
/*
== Tittle
-- Description
## Content
*/
```
